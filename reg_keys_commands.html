Command to see if there is any corrupted windows components:


cmd /c DISM /Online /Cleanup-Image /Scanhealth > c:\$VcmsTemp$\logs\dismhealth.log


FYI - the process can last around 15 mins, so please while it checks everything



cmd /c del "C:\$Vcmstemp$\Logs\*.*" /F /Q - to clear all data in logs
