UBUNTU

cat /etc/os-release - returns information on OS installed on a device locale - returns info on configured language settings hostname - returns the name which was set on a PC, Not FQDN (for LQDN need to add flag -f) who -b - shows the logged user. dmidecode | grep -A3 '^System Information' - DMI - special tool which allows to return info on a hardware of PC. ifconfig - utility which allows to manage network interface configurations cat /proc/version - returns version of Ubuntu and core installedm date when it was installed ip route | grep default - returns the default ip route from ip route table

UBUNTU additionall commands

These commands can be useful: touch /path/file - creates file in in specified directory locale -a - display the available locales locale-gen fr_FR.UTF-8 - if locale doesn't appear in list, it should be installed additionally dpkg -l - returns the list of all packages
